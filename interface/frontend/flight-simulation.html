info Como conectar o FlightGear

  • 1. Inicie o Backend do FlightGear python fg-bridge.py .
  • 2. Inicie o FlightGear com a opção Telnet/WebSocket apontando para --telnet=socket,bi,60,localhost,5050,tcp .
  • 3. Este serviço limita automaticamente os ângulos a ±12° e força z = 540 mm.
  • 4. Só após o botão “Iniciar seguimento” a pose é aplicada no backend app.py (FastAPI).

Bridge ativo em ws://localhost:9001 .

playlist_add_check Procedimento seguro

  1. Conecte a placa ESP32-S3 e confirme a telemetria.
  2. Use “Ir para z seguro” para posicionar a plataforma em 540 mm.
  3. Clique em “Iniciar seguimento” para liberar o bridge.
  4. No FlightGear, envie roll/pitch via WebSocket para testar.
  5. Finalize clicando em “Parar seguimento”.
